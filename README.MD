# Project Name

**Connect 4 Game**

## Technologies Used

- HTML
- CSS
- JavaScript

## Description

A simple implementation of the classic **Connect 4** game where two players take turns dropping colored discs into a vertical grid. The first player to connect **four discs** horizontally, vertically, or diagonally wins the game.

---

## User Stories

- As a Player I want to see 5 x 5
- As a Player should pick a color to play Between yellow - red
- As a player I Should go first
- Marks alternate between Player and Computer after each valid turn.

IN WIN Condition :

- As A Player I want to be Notified immediately when a Player Aligns 4 of the Marks in a Row , Column Or Diagonal.

IN Tie Condition :

- As A Player I want to be Notified Immediately When All Spaces get chosen and no one connects 4 in a Row , Column Or Diagonal.
